Aplikacija Klepetalko bo uporabnikom omogočala dodajanje novih prijateljev in medsebojno komunikacijo (chat app). Uporabnik bo lahko naredil 
skupinski pogovor z več uporabniki skaterimi je sklenil prijateljstvo. V sporočilih bodo uporabniki lahko pošiljali tudi priponke.
